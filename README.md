
Este programa crea un arreglo de 1,000,000 de elementos y permite buscar un número utilizando:
- Búsqueda secuencial
- Búsqueda binaria

Funcionamiento:
El usuario ingresa un número por teclado.
El programa busca el número utilizando ambos métodos y muestra:
- Si fue encontrado
- La posición
- El tiempo de ejecución

Comparación:
La búsqueda secuencial revisa elemento por elemento, por lo que tarda más tiempo.
La búsqueda binaria divide el arreglo en mitades, siendo mucho más rápida cuando el arreglo está ordenado.
